nama aplikasi : pendaftaran mahasiswa baru dengan AI
deskripsi : mahasiswa yang bingung untuk memilih jurusan sesuai keahliannya, maka aplikasi ini jawabannya. yang dimana aplikasi ini dibuat untuk memilih jurusan mahasiswa yang bingung
fitur : halaman login, user memilih ingin jurusan apa yang diinginkan lalu user akan di arahkan ke testing, lalu akan di tes psikologi dan ai akan menilai lulus/tidaknya user dan kalau luus akan direkomendasikan jurusan yang sesuai keahliannya
cara menjalankan : extrack zip, tambahkan/import database, jalankan backend dan frontend
